# Java-R201

Java OOP exercises for the university course R201.

## Structure

- **TP1** — HelloWorld, Point class, Student class
- **TP2** — Bounded integer sets, perfect squares
- **TP3** — Segments, garage (cars/wheels/engine), florist (bouquets/flowers), boxes
- **TP4** — Equation solver, argument summing, detection
- **TP5** — Inheritance and polymorphism: shapes, colored points, vehicles, library catalog

Each TP folder contains a `src/` directory with Java sources and a PDF assignment sheet.

## Build & Run

```bash
cd TP1/src/helloWorld
javac HelloWorld.java
java HelloWorld
```
