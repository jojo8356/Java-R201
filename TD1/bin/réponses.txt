Astuces:

Compiler 1 seul fichier et l'exécuter:
javac -d bin NomFichier.java
java -cp bin nomPackage.NomClasse

Compiler 1 fichier avec 1 dépendance (autre package) et l'exécuter:
javac -cp ../point/bin -d bin TestPoint.java
java -cp ../point/bin:bin point2.TestPoint

Compiler plusieurs fichiers d'un coup:
javac -d bin *.java

Compiler tout un projet depuis la racine:
javac -d bin package1/*.java package2/*.java

Rappels:
- -d bin : répertoire de destination des .class
- -cp ou -classpath : chemin vers les classes dont on dépend
- Sur Linux/Mac, séparer les chemins avec ":"
- Sur Windows, séparer les chemins avec ";"
